# DigiTools Platform

A modern React-based digital product platform showcasing premium AI tools, product catalog, cart management, responsive design, and interactive UX.

## Description

DigiTools is a fully responsive web application that allows users to browse digital product offerings, add items to a cart, and complete checkout actions with animated interactions and toast notifications.

### What it supports

- Product cards with icon, description, price, tags, and features.
- Cart section with add, remove, and checkout actions.
- Navbar cart count and view switching.
- Strictly responsive across devices.
- Toast feedback via `react-toastify` for each action.

## Technology Stack

- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- React Toastify

## 3 Key Features

1. Product / Cart toggle, default product view.
2. Add/Remove products in cart and proceed-to-checkout clear.
3. Responsive UI, hero/stats/steps/pricing/footer sections.
